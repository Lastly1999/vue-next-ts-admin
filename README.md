# Vue3+TypeScript+Ant Design学习项目
* 学习vue3.0以及日常使用组件开发的一个demo

# 技术栈
* 前端:vue3.0 Ant Design2.9Rc UI框架库 TypeScript3.9 antV 数据可视化
* 后端:Java Springboot 验证模块 java-jwt身份验证 数据库 MYSQL8.x

# 后台账号权限管理
* 菜单权限管理
* 后台身份权限管理
* 菜单管理

# 组件模块
* 公用导航菜单
* 公用表格
* 公用表单
* 公用树
* 富文本编辑器
* 数据仪表盘
* 地图组件(高德地图)
* 验证码验证模块

# 其他功能
* 直播视频流(在线录制屏幕)
* 聊天室(WebSoket)
